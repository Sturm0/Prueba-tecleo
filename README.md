Dependencias: ncursesw


En Ubuntu debería alcanzar con:


```sudo apt install libncurses-dev libncursesw6```

## Instrucciones para instalar y usar
Descargar el ".tar.gz" de "Releases", extraerlo, ejecutar (en un terminal) `make`

Eso debería dejar un ejecutable con nombre "Main"

Desde un terminal ejecutar Main usando "./Main"

Para salir del programa escribir "salir" (sin las comillas).
